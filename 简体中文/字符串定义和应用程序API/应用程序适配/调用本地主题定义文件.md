# 使用本地定义文件

你的应用程序也可以使用一个本地的定义文件。设定了本地定义文件后，当前主题数据中没有一个字符串定义时会尝试本地定义文件中的数据。一般情况下，你应该把本地定义文件放在应用程序中的资源文件或代码文件内。本地定义文件可以用于适配多个显示语言（类似于GNU gettext）和其他类似用途。

## 设定本地定义

设定本地定义文件可以使用`frontend.set_local_themedef`函数，并且把文件的内容传给这个函数。当设定成功时，该函数会返回`True`，否则会返回`False`。

```py
from clitheme import frontend
filename = #...
content=open(filename, "r", encoding="utf-8").read()
frontend.set_local_themedef(content)
```

设定完成后会对同一个Python模块中所有文件的字符串调用操作生效。

## 同时设定多个本地定义文件

你可以通过`frontend.set_local_themedefs`函数同时设定多个本地定义文件。使用时，请通过列表指定多个文件内容。除此之外，你可以使用`set_local_themedef`函数并设定`overlay=True`参数。

```py
from clitheme import frontend
# ...
frontend.set_local_themedefs([file_content1, file_content2, file_content3])
# 也可以使用以下方法：
frontend.set_local_themedef(file_content1)
frontend.set_local_themedef(file_content2, overlay=True)
frontend.set_local_themedef(file_content3, overlay=True)
```

## 取消设定

如需取消设定本地定义文件，请使用`frontend.unset_local_themedef`函数。取消设定后，frontend将不会尝试调用本地定义文件中的字符串定义。

```py
frontend.unset_local_themedef()
```
