# 内容变量的定义和使用

你可以在主题定义文件中定义和使用**内容变量**，可以让修改和使用重复内容更方便。

## 定义变量

如需定义变量，请使用`setvar[<变量名称>]: <内容>`语法。你也可以同时指定多个由空格分开的变量名称，如`setvar[var1 var2]:`。

```plaintext
{substrules}
    setvar[example_var]: This is some example content
    # 同时设定var1和var2变量为"Another example variable"
    setvar[var1 var2]: Another example variable
    # <...>
{/substrules}
# 你也可以在段落之外定义变量（全局定义）
setvar[another_var]: Some variable content
setvar[var3]: Third variable!
```

> [!NOTE]
> 变量名称不能包含`(){}[]`中的任一字符，并且不能为`ESC`（因为会和`substesc`功能选项冲突）。

### 全局定义和段落内定义的区别

在全局定义是指在段落外定义的变量。**段落内定义的变量只会在该段落内有效，包括对全局变量的修改**。结束该段落后，当前的变量定义将会复原成现有的全局定义。

## 使用变量

如需使用变量，请先[启用](./可以设定的选项.md)`substvar`选项，然后在内容中使用`{{<变量名称>}}`语法，比如`{{example_var}}`。

```plaintext
setvar[name]: Vim
(set_options) substvar
{header}
    name: {{name}} theme
    description: A simple theme for {{name}}
{/header}
{substrules}
    [subst_string] {{name}} 9.1
        default: {{name}} Improved 9.1+
    [/subst_string]
{/substrules}
```

你也可以在指定系统语言和选项设定的地方使用内容变量，比如：

```plaintext
{substrules}
    (enable_subst)
    setvar[langs]: en_US C
    setvar[opts]: strictcmdmatch endmatchhere
    [subst_regex] Some content
        # 等于"locale[en_US C]:"
        locale[{{langs}}]: Substitute content
        # 等于"[locale] en_US C"
        [locale] {{langs}}
            Substitute content
        [/locale]
    # 设定`strictcmdmatch`和`endmatchhere`选项
    [/subst_regex] {{opts}}
{/substrules}
```

需要注意的是，变量不能替代语法定义。这意味着你不能使用以下语法：

```plaintext
(set_options) substvar
setvar[place1]: name:
setvar[place2]: version:
{header}
    # 错误用法：
    # {{place1}} Vim theme
    # {{place2}} 9.1
{/header}
```

> [!NOTE]
> - 变量名称中不能包含任何空格，否则将不会被处理。比如：`{{var }}`和`{{this and}}`
> - 变量内容的ESC或字符替换表达式**不会在定义变量时被处理**；只有变量在支持的内容中被引用时才会被处理

## 在变量定义中引用另一个变量的内容

你可以在变量的定义内容中通过上述方法引用另一个变量的内容。前提是：在定义变量之前设定`substvar`选项。

```plaintext
setvar[example_one]: Some text here
# 启用substvar之前，将不会替换变量名称
# 内容：{{example_one}} with another text
setvar[example_two]: {{example_one}} with another text
(set_options) substvar
# 启用substvar之后将会替换变量名称
# 内容：Some text here with another text
setvar[example_three]: {{example_one}} with another text
```
