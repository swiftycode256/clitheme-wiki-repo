# Providing necessary information to users

To let users know the customizable string entries and messages, your application need to provide description or documentation of them. Each entry should provide the path name and default message.

For example, you can list the following in your documentation:

```plaintext
com.example example-app found-file
[default] Found {} files in current directory
[zh_CN] 在当前目录找到了{}个文件

com.example example-app installing-file
[default] Installing {}...
[zh_CN] -> 正在安装"{}"...

com.example example-app install-success
[default] Successfully installed {} files
[zh_CN] 已成功安装{}个文件

com.example example-app file-not-found
[default] Error: can't find file {}
[zh_CN] 错误：找不到文件 "{}"
```
